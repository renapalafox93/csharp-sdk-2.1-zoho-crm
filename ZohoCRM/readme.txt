Zoho CRM C# SDK 1.0.0
————————————————————————

This is the readme file for Zoho CRM’s C# SDK version 1.0.0.

This file gives a brief of the enhancements and/or bug fixes in the latest version.

----------------
Enhancements
----------------
    - C# SDK upgraded to support v2.1 APIs.
    - Structural changes to aid the process of SDK configuration and initialization user-friendly.
    - C# SDK improved to support the following new APIs
        - Assignment Rules
        - Email Templates
        - Field Attachments
        - Inventory Templates
        - Pipeline
        - Send Mail
        - Wizards

You can also take a look at our GitHub page here (https://github.com/zoho/zohocrm-csharp-sdk-2.1/blob/master/README.md)
