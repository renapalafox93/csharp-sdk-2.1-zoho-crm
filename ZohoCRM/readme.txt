————————————————————————

This is the readme file for Zoho CRM’s C# SDK version 4.0.0-beta.

This file gives a brief of the enhancements and/or bug fixes in the latest version.

----------------
Enhancements
----------------
    - The SDK version adopts the Zoho CRM v2.1 APIs
    - The SDK is structured to be more user-friendly

The basic usage pattern has not changed from version 3.

You can also take a look at our GitHub page here (https://github.com/zoho/zohocrm-csharp-sdk-2.1/blob/beta/README.md)
